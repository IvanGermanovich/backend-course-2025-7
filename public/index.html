Quick API Reference:

  • POST /register - Register a new item with photo
  • GET /inventory - Get all items
  • GET /inventory/:id - Get specific item
  • PUT /inventory/:id - Update item details
  • DELETE /inventory/:id - Delete item
  • GET /inventory/:id/photo - Get item photo
  • PUT /inventory/:id/photo - Update item photo
  • POST /search - Search for item by ID
